请求数据包:

POST /ddddocr/base64 HTTP/1.1
Host: x.x.x.x:x
Content-Type: application/json

{
    "user": "user",
    "token": "token",
    "image": "<@URLENCODE><@BASE64><@IMG_RAW></@IMG_RAW></@BASE64></@URLENCODE>"
}

返回数据包:

HTTP/1.0 200 OK
Content-Type: application/json
Content-Length: 31
Server: Werkzeug/2.0.1 Python/3.7.2
Date: Wed, 02 Mar 2022 03:22:13 GMT

{"result":"slrr","status":"1"}

参数说明

字段名 类型 解释
user string 用户名
token string 身份认证标识，个人信息页面获取。
image string 图片原图像base64编码
result string 返回的识别结果
status string 是否识别成功，成功返回1，失败返回0
